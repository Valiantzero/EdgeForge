**EdgeForge** is an experimental crypto strategy research lab focused on testing, validating, and monitoring trading setups before they are ever considered for live execution.

The project is currently in **private beta** and under active development.

EdgeForge is not marketed as a “money-making bot”.  
It is a research and validation platform designed to help reject weak strategies, compare market conditions, and move only validated setups into paper or testnet environments.

---

## What EdgeForge Does

EdgeForge helps research trading strategies through a structured workflow:

1. Build or select a strategy configuration
2. Backtest it over historical candle data
3. Validate performance across rolling time windows
4. Compare results across market sessions
5. Run controlled parameter sweeps
6. Promote only validated setups to paper/testnet
7. Export results for external review and analysis

The main goal is not to find the highest Profit Factor in one backtest.

The goal is to identify setups that remain reasonably stable across time, sessions, and validation windows.

---

## Current Focus

The current beta is focused on BTCUSDT futures research using Binance testnet/paper execution.

Main components currently being developed:

- Historical backtesting
- Overnight strategy search
- Edge validation
- Session comparison
- Controlled parameter sweep
- Rolling validation
- Paper/testnet deployment
- Strategy preset management
- Export packages for external analysis
